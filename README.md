# Uygar Çatal — Interactive CV

A single-file, self-contained interactive CV/résumé page (HTML + CSS + JS, no build step).

🔗 **Live:** https://uygarcatal6.github.io/

## Features
- Expandable cards (click / keyboard), reveal-on-scroll, print-to-PDF styling.
- Responsive (desktop / tablet / mobile) and accessible (keyboard, ARIA, reduced-motion).
- No dependencies beyond Google Fonts + Material Symbols (loaded via CDN).

## Local preview
Just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8099
# then visit http://localhost:8099
```
